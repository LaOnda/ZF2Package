bootstrap( \Zend\Mvc\Application $application ) : void

Bootstrap the application

  • Initializes the locator, and injects it in the application
  • Initializes the router, and injects it in the application
  • Triggers the "bootstrap" event, passing in the application and modules as parameters. This allows module classes to perform arbitrary initialization tasks after bootstrapping but before running the application.
